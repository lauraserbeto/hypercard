# HyperCard — O Clube Financeiro da Elite

**HyperCard** é uma plataforma bancária fictícia de ultra-luxo, projetada para oferecer uma experiência digital sem precedentes para executivos, empresários e investidores de alta renda. O projeto combina uma estética minimalista e "Dark Mode" com interações fluidas e uma arquitetura multi-página robusta.

![HyperCard Preview](https://github.com/user-attachments/assets/65480-placeholder-style-hero) 
## ✨ Destaques do Projeto

*   **Design Premium & Imersivo**: Uso de *glassmorphism*, gradientes sutis e efeitos de *backdrop-blur* para uma sensação de exclusividade.
*   **Animações 3D**: Showcase de cartões com movimentação 3D interativa e efeitos de "fan spread" utilizando Framer Motion.
*   **Arquitetura Multi-página**: Sistema completo de rotas usando Next.js App Router, incluindo páginas institucionais dedicadas.
*   **Experiência Mobile-First**: Layout totalmente responsivo, adaptando a sofisticação da elite para qualquer dispositivo.
*   **Performance**: Otimização de imagens e carregamento inteligente de componentes.

## 🛠️ Tecnologias Utilizadas

*   **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
*   **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animações**: [Framer Motion](https://www.framer.com/motion/)
*   **Ícones**: [Lucide React](https://lucide.dev/)
*   **Tipografia**: Google Fonts (Inter & Serifadas)
*   **Linguagem**: TypeScript

## 📂 Estrutura do Site

O projeto foi expandido para uma estrutura institucional completa:

*   **Home**: Landing page com Hero 3D, seção de App Mobile e FAQ.
*   **Sobre Nós**: Manifesto da marca, timeline histórica e apresentação da liderança executiva.
*   **O Cartão**: Detalhamento técnico e visual dos benefícios dos cartões Platinum, Black e Gold.
*   **Segurança**: Hub de informações sobre criptografia pós-quântica e privacidade.
*   **Concierge**: Apresentação dos serviços de lifestyle management 24/7.
*   **Compliance & Legal**: Páginas de regulamentação, termos de uso e transparência institucional.

## 🚀 Como Executar

1.  **Clone o repositório**:
    ```bash
    git clone https://github.com/seu-usuario/hypercard.git
    ```

2.  **Instale as dependências**:
    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```

4.  **Acesse no navegador**:
    `http://localhost:3000`

## 🎨 Identidade Visual

A paleta de cores foca no preto absoluto (`#000000`) com acentos em branco semi-transparente e tons metálicos (ouro e platina), criando um contraste de alto nível que define o padrão HyperCard.
